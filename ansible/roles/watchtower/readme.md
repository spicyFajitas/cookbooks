# Watchtower

## Discord Webhook Alerts

Your Discord Webhook-URL will look like this:

`https://discord.com/api/webhooks/webhookid/token`

The shoutrrr service URL should look like this:

discord://token@webhookid

Example:

Standard Discord link:

`https://discord.com/api/webhooks/12234567890xxxxxxxxx/AbCdEfGhIjAbCdEfGhIjAbCdEfGh-AbCdEfGhIjAbCdEfGhIjA-AbCdEfGhIj7-AbCdEf`

`discord://AbCdEfGhIjAbCdEfGhIjAbCdEfGh-AbCdEfGhIjAbCdEfGhIjA-AbCdEfGhIj7-AbCdEf@12234567890xxxxxxxxx`
